# B站专栏文章发布指南

## 一、发布前准备

### 1. 登录 B站账号

访问 https://member.bilibili.com/ 并登录你的 B站账号

### 2. 进入专栏创作中心

点击右上角「创作中心」→「专栏」

或直接访问：https://member.bilibili.com/platform/article

---

## 二、发布文章步骤

### 步骤 1：点击「发布专栏」

### 步骤 2：填写文章信息

| 项目 | 填写内容 |
|------|---------|
| **标题** | 在 Android 手机上运行完整 Linux 系统！Alpine Linux Magisk 模块使用教程 |
| **分类** | 科技 → 数码/软件 |
| **封面** | 建议上传一张 1146×717 的封面图 |

### 步骤 3：编辑正文

1. 将 `BILIBILI_ARTICLE.md` 文件内容复制
2. B站专栏支持富文本编辑器，可以直接粘贴

**注意格式调整：**

| 原格式 | B站格式 |
|--------|--------|
| `# 标题` | 使用标题样式（H1/H2） |
| `**粗体**` | 使用粗体按钮 |
| ` ```代码``` ` | 使用代码块样式 |
| 表格 | 使用表格功能或截图 |

### 步骤 4：添加标签

推荐标签（最多5个）：
- Android
- Linux
- Magisk
- 教程
- 技术

### 步骤 5：设置投稿选项

- **原创/转载**：选择「原创」
- **允许评论**：开启
- **允许转载**：开启

### 步骤 6：点击「发布」

---

## 三、B站专栏格式注意事项

### 1. 标题限制

- 2-50 个字符
- 不能包含特殊符号
- 建议使用吸引眼球的标题

### 2. 正文限制

- 最多 20000 字
- 图片单张最大 5MB
- 支持 jpg、png、gif 格式

### 3. 推荐做法

| 项目 | 建议 |
|------|------|
| 封面图 | 制作一张有吸引力的封面 |
| 代码块 | 使用代码块样式，添加语法高亮 |
| 图片 | 添加操作截图，增加可读性 |
| 分段 | 每段不宜过长，方便阅读 |

### 4. 禁止内容

- 政治敏感内容
- 广告推广
- 侵权内容
- 虚假信息

---

## 四、文章优化建议

### 1. 添加操作截图

建议在以下位置添加截图：
- 安装成功截图
- 运行状态截图
- SSH 连接截图

### 2. 制作封面图

封面图建议包含：
- 项目 Logo 或 Alpine Linux 标志
- 吸引人的标题文字
- 简洁的背景

推荐尺寸：1146×717 像素

### 3. 互动引导

文章末尾添加：
```
觉得有用的话，请：
- 点赞支持
- 收藏备用
- 关注作者
- 评论区交流
```

---

## 五、发布后操作

### 1. 检查审核状态

发布后文章会进入审核，一般 1-24 小时内完成

### 2. 回复评论

及时回复读者评论，增加互动

### 3. 分享推广

分享到：
- 相关技术群
- 知乎
- 掘金
- 公众号

---

## 六、常见问题

**Q: 文章审核不通过？**

A: 检查是否包含敏感词、广告、侵权内容

**Q: 图片上传失败？**

A: 检查图片大小和格式，建议压缩后上传

**Q: 代码块显示异常？**

A: 使用 B站编辑器的代码块功能，或截图展示
